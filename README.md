# Ejercicio 2 - Formulario + Supabase

Autor: Gonzalo Vico Marin

## Descripción
Se ha desarrollado un formulario en Liferay 7.4 con validaciones que permite guardar datos en una base de datos en la nube usando Supabase y mostrar un listado de solicitudes.

## Funcionalidades
- Validación de campos
- Envío de datos
- Guardado en base de datos cloud
- Listado de registros

## Desarrollo
El ejercicio se ha realizado sin uso de IA.  
Me ha costado bastante entender la conexión con Supabase y el funcionamiento de Liferay.  
He tardado aproximadamente 3 días en completarlo.
